# bussinessEmployee

ChatingAppUsingSignalR is a simple web application (MVC) for CRUD operations .

## Features:
- CRUD operations on Departments and employees
- Localization / Globalization
- search 
- Filter
- Pagination
- Registration 
- authentication  
- authorization (Role Based)
- Forget Password
- Reset Password

## Tools :       
* C#
* MVC
* EF(Code First)
* repository Pattern
* View Model Binding
* Identity
* DataTable plugin
* Bootstrap
* javascrpit
* JQuery
* AJAX
